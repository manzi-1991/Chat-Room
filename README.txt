第六步，核心步骤，分两部分互相对应。
	由服务器端 var io = require('socket.io')(http);  引出  浏览器端静态文件资源/socket.io/socket.io.js

	服务器设置io.on("connection",function(socket){｝    浏览器设置<script type="text/javascript" src="/socket.io/socket.io.js"></script>
									<script type="text/javascript"> var socket = io();

	此时，服务器端和客户端均有socket对象，都拥有emit和on方法。emit方法发送自定义事件和内容，on方法监听自定义事件和内容，内容作为回调参数
	从而产生实时交互效果。
